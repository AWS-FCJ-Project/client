🏁 Bắt đầu (Getting Started)
Để chạy dự án ở môi trường local, hãy làm theo các bước sau:

1. Clone dự án

Bash
git clone https://github.com/your-username/edutrust.git

cd edutrust
3. Cài đặt thư viện
   
Bash
npm install

# hoặc
yarn install

3. Chạy môi trường Development
   
Bash
npm run dev

Mở http://localhost:3000 trên trình duyệt để xem kết quả.
